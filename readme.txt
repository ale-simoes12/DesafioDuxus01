Instruções:

O banco de dados utilizado foi o PostgreSQL. Para integrá-lo ao programa, foi criada uma classe conexão no método getConnection(). É necessário fornecer login e senha do banco para que funcione. Além disso, caso queira usar outro banco, é necessário modificar o arquivo pom.xml.

Entrada:

Cada classe (Integrante, Time, ComposicaoTime) possui métodos para inserir valores nas tabelas do banco.

Métodos:

Os métodos da classe ApiService não estão completos.
